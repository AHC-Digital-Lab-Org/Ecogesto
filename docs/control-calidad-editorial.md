# Control de Calidad Editorial de EcoGestos

## Estados editoriales

| Estado | Quien actua | Criterio |
|---|---|---|
| Borrador | Autor o voluntario | Contenido minimo creado |
| En revision | EcoGestor o I+D | Se revisa coherencia, fuentes, categoria, imagen y licencia |
| Pendiente de calculo | Equipo tecnico/ambiental | Falta formula o factor suficiente |
| Validado | Responsable AHC | Listo para publicacion interna o publica |
| Publicado | Administrador/EcoGestor | Visible en catalogo |
| En revision periodica | EcoGestor | Caducidad, nuevas fuentes o comentarios |
| Archivado | Administrador | Oculto, conserva historico |

## Ficha minima publicable

- Codigo unico.
- Nombre accionable.
- Resumen claro.
- Categoria y subcategoria.
- Perfil recomendado.
- Dificultad.
- Coste.
- Impacto estimado o estado pendiente.
- Periodo.
- Guia paso a paso.
- Evidencia requerida.
- Fuente o nota de estimacion.
- Icono o imagen.
- Estado Moodle.
- Licencia, autor y texto alternativo si hay imagen.

## Checklist de revision editorial

- El nombre empieza con verbo o accion clara.
- El resumen no culpabiliza.
- La guia tiene pasos ejecutables.
- No se prometen ahorros sin fuente.
- Las barreras principales tienen solucion o variante.
- El coste distingue inicial y ahorro.
- La dificultad refleja cambio de habito real.
- La fuente no es vaga si el impacto se muestra como validado.
- El tono respeta "cada gesto cuenta" y "mejor progreso que perfeccion".
- No hay emojis en interfaz ni textos.

## Revision periodica

Frecuencia recomendada:

- Alta confianza: anual.
- Media confianza: cada 6 meses.
- Baja o pendiente: no publicar como validado.
- Factores energeticos: revisar al menos una vez al año.
- Normativa/residuos/RGPD: revisar ante cambios legales.

## Exportacion para revision

La matriz `docs/matriz-validacion-ecogestos.csv` se puede abrir en Excel/Sheets y asignar:

- Responsable ambiental.
- Responsable editorial.
- Estado de formula.
- Estado de fuente.
- Estado de Moodle.
- Fecha de revision.
- Observaciones.
